三、核心原理
De Casteljau 算法
输入一组控制点 P0,P1,…,Pn-1，参数 t 在 0 到 1 之间。对相邻点递归进行线性插值，公式为 P'=(1-t)P0 + tP1，不断迭代直到只剩下一个点，该点即为曲线上的点。
光栅化
屏幕使用 800x800 像素数组模拟。将归一化坐标 0 到 1 之间的数值映射为 0 到 800 的整数像素坐标，对对应坐标赋值颜色即可点亮像素。
反走样
对亚像素坐标周围 3x3 邻域根据距离加权着色，有效消除曲线锯齿，实现平滑边缘效果。
三次均匀 B 样条
具备局部控制特性，曲线更加平滑稳定。每 4 个控制点生成一段曲线，多段之间自动平滑拼接。
